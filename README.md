# Podcast Studio — 內容工廠

以「兩人自然閒聊式對話」為核心，用 AI 放大產能、降低後製摩擦，把
**發想 → 錄製 → 產出 → 後製 → 推廣** 整合成穩定的內容生產線。

## 核心理念

> 自然閒聊對話是「原料」，AI 是「放大器」，不是取代者。

- 真人不必出鏡（視覺生成）
- 不怕錄製卡詞（先輕鬆討論，再用克隆聲音依腳本生成流暢對話）
- 所有 AI 環節保留「人在迴路」的覆核點，守住內容正確與品質
